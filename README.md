# Сервис бухгалтерских справок

Тестовое задание для автоматизации процесса запроса справок сотрудниками.

## Функциональность

- **Для сотрудников:**
  - Подача запросов на справки (2-НДФЛ, стаж и др.).
  - Просмотр статуса своих запросов.
  - Защита от случайных дубликатов (ограничение на повторную отправку идентичного запроса в течение 5 минут).

- **Для бухгалтеров:**
  - Единая очередь всех заявок.
  - Просмотр подробностей: ФИО сотрудника, тип справки, количество копий, причина.
  - Управление статусами (В работу, Готово, Отклонить).

## Технологии

- **Backend:** .NET 9 Web API.
- **Frontend:** HTML5, CSS (Pico.css), Vanilla JavaScript.
- **База данных:** PostgreSQL (через Entity Framework Core).
- **Контейнеризация:** Docker Compose для БД.

## Как запустить

1. Убедитесь, что у вас установлены .NET 9 SDK и Docker Desktop.
2. Запустите базу данных:
   ```bash
   docker-compose up -d
   ```
3. Соберите проект: `dotnet build`
4. Запустите проект: `dotnet run --project TestTaskRosa`
5. Откройте браузер по адресу: `http://localhost:5272` (или порту, указанному в консоли).

## Настройка БД
Параметры подключения настроены в `appsettings.json`. По умолчанию используется:
- Host: localhost:5436
- DB: rosa_db
- User: postgres
- Password: postgres
